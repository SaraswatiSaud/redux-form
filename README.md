# redux-form

## Installation
```
1. git clone https://github.com/SaraswatiSaud/redux-form.git
2. cd redux-form
3. npm run setup
4. npm run dev
5. open http://localhost:3000
```

## Folder Structure

After creation, your project should look like this:

```
redux-form/
  node_modules/
  public/
    index.html
    favicon.ico
    src/
      components/
      css/
      index.js
      registerServiceWorker.js
  README.md
  package.json

```
